How to use the result

  1. Enter the actual dose received by this organ during both treatment courses, not merely the prescription dose.
  2. Select the Timmerman row that matches the proposed fraction schedule and applicable volume or point-dose metric.
  3. A green result means the proposed second-course dose is within that selected Timmerman single-course limit.
  4. A red result means it exceeds the selected Timmerman limit. The cumulative result is shown only as an additional warning because Timmerman is not a reirradiation tolerance table.
